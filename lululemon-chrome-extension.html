Lululemon Chrome Extension

I've always enjoyed running, but it wasn't until my car unexpectedly died last April while looking for apartments in Philadelphia that I started running several miles on a weekly basis. If I have an eye appointment in center city, I'll run there. If I'm meeting friends at an outdoor orchestra concert in Old City, I'll run there. Thanks to Spotify, Lululemon, and the "always sunny" Philly weather, running has become one of my favorite things to do in my free time, and it's a great way to explore the city.

Run: Inspire Crop II at full price.

I'm quite fond of a particular type of running crops from Lululemon, but at nearly $100 a pop, my bank account doesn't share the same affection for them. Whenever I remember, I visit the online Lululemon sale site and scroll through all the items to see if my crops are there (or command+f if I remember the item names). Before arriving at the Recurse Center, I toyed with the idea of building a system that would somehow alert me when my favorite crops go on sale online. However, I wasn't sure what steps would be involved (would I need to set up a Heroku server to run a script once a day?) or where to start. When I got to RC, a few people were building Chrome extensions and I knew immediately this was exactly the way to go for this project.

Before building my extension, I'd manually scan through the sale looking for my running crops.

I started following this tutorial, which did a decent job of teaching me about the manifest file and loading my extension for development. Looking back at this tutorial now, I'm realizing I skipped the "Learn Basics" section which would have been useful.

Everything about working with the web, browsers, and javascript was new to me. I had never heard of requests, events, or DOMs. A callback is what an actor or musician receives when he/she makes it to the next round of an audition. It's also a thing in javascript that is much harder to understand but mission critical to my extension! Callbacks allow code to be executed in the order that corresponds to how the client is interacting with the webpage.

When I click on the lululemon icon, the javascript in my Chrome extension executes. A box appears showing which of my prespecified items are on sale and displaying all the available sale colors.
