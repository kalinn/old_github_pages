Pinpoint: My First Flask App

With just two weeks left until never graduating from the Recurse Center, I'm trying to polish and publish my code as well as document some of the lessons learned from each of my projects. Pinpoint is a simple web app built using Flask, a web microframework based in Python. Last week, I deployed it on AWS's Elastic Beanstalk, and anyone with an Instagram account can play the city guessing game here. Source code is available on github here.

Despite its simplicity, building Pinpoint introduced me to a range of topics beyond Flask web development, including: - Working with Instagram's API - How to keep a secret (that is, a secret key) - Web app deployment with Amazon Web Services

Lessons learned: Instagram's API

Instagram's API is very nice and can be conveniently accessed using the python-instagram library.

Lessons learned: how to keep a secret

After accidently publishing my Instagram client ID and secret key to github, I decided it was finally time to learn git beyond the add, commit, and push commands. My new best friends are git status and .gitignore files. Environment variables are cool too.
